How it works
1
Enter your API key
Paste your Anthropic API key below. It stays only in this browser — never sent anywhere except directly to Anthropic.

Upload Excel or PDF
Drop any file — invoice list, salary sheet, expense register, compliance tracker. Any format, any column names.
3
AI reads & maps
Claude reads the file, detects which module(s) the data belongs to, and extracts all records intelligently.

Review & confirm

📤 Share & Sync Data

✅Data saved locally in this browser.
💾
Export as JSON file Recommended
Downloads a single .json file containing ALL your data — invoices, expenses, tasks, compliance, payroll, everything.
Share this file with anyone. They open the tracker HTML + import this file to get your exact data.
📂
Import from JSON file Load data
Open a previously exported .json file to load that data into this tracker.
⚠ This will replace your current data. Export first if you want to keep it.
🔗
Share via Link View-only snapshot
Generates a URL with all your data encoded inside it. Send via WhatsApp or email — recipient opens in browser and sees your exact data. Note: Large datasets may produce very long URLs.
📱 How to use on phone
Step 1: Export the JSON file from any device where you have data.
Step 2: Send the HTML file + JSON file to your phone (WhatsApp, email, Google Drive).
Step 3: Open the HTML file in Chrome on your phone → tap Import → select the JSON.
Or: Use "Generate share link" above and open the link directly on your phone.
